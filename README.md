## optitrack_person

small [ros](wiki.ros.org) package to have [optitrack](http://www.optitrack.com/) data in [hanp_msgs](https://github.com/harmishhk/hanp_msgs) format

### build

- install dependencies
  - ```ros-indigo-deksop```
  - [*optitrack-genom3*](https://git.openrobots.org/projects/optitrack-genom3) module installed from [robotpkg](http://robotpkg.openrobots.org/)
- run ```catkin_make install```
